# Tecnologia para Frontend com Lógica Imperativa

Este repositório contém minhas submissões para as listas de exercício para
disciplina de Tecnologia para Frontend com Lógica Imperativa da instituição
UNIESP.

## Requisitos

- Node.js v22

## Estrutura do Repositório

As submissões para as listas se encontram na pasta `src`. Cada pasta dentro
de `src` corresponde a uma lista de exercícios.

## Executando os Exercícios

Para executar os exercícios, basta executar o arquivo da submissão usando o
comando `node`.

**Exemplo:**

```bash
node src/03-json/index.js
```
